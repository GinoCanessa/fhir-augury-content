## Description

The reporter suggests improving the readability and navigability of the DICOM SR to FHIR Resource Mapping IG by:

1. **Linking subsections of Section 4 (Mapping)** to the Mappings defined for every profile — so readers can easily navigate between the narrative mapping descriptions and the formal profile-level mappings.
2. **Aligning the content** between Section 4's subsections and the per-profile Mappings tabs to ensure consistency and reduce confusion.

This is an editorial/usability improvement request, not a technical change to the mappings themselves.

## Proposed Dispositions

### Disposition A: Persuasive — Add Cross-Links and Align Content

**Action:** Accept the request in full. Add hyperlinks from each Section 4 subsection to the corresponding profile's Mappings tab, and audit/align the narrative mapping text with the formal profile mappings to ensure consistency.

**Rationale:** This is a straightforward editorial improvement that enhances IG usability without changing any technical content. The reporter correctly identifies that readers must manually cross-reference Section 4 with profile-level mappings, which is error-prone and confusing. Adding links is low-effort and high-value. Aligning the content addresses a real consistency gap that multiple related tickets (FHIR-47673, FHIR-47384, FHIR-47377) also highlight.

**Impact:** Editorial changes to Section 4 narrative pages and possibly profile pages; no changes to profiles, value sets, or mapping definitions themselves.

### Disposition B: Persuasive with Modification — Add Cross-Links Only, Defer Full Alignment

**Action:** Add cross-links from Section 4 subsections to profile Mappings tabs, but defer the full content alignment audit to a separate effort that addresses FHIR-47673, FHIR-47384, and FHIR-47377 together.

**Rationale:** The cross-linking request is unambiguous and can be done immediately. The "align content" request is broader and overlaps significantly with FHIR-47673 (Mapping vs. Example alignment) and FHIR-47384/47377 (graphical representation alignment). Addressing alignment holistically across all related tickets would be more efficient and produce a more consistent result.

**Impact:** Smaller initial editorial change (links only); a follow-up task would handle the larger alignment effort.

### Disposition C: Not Persuasive — Existing Navigation is Sufficient

**Action:** Decline the request. The IG's existing structure (table of contents, profile tabs) already provides sufficient navigation between Section 4 and profile mappings.

**Rationale:** The FHIR IG publisher already generates a standard table of contents and profile pages with Mappings tabs. Adding additional cross-links may create maintenance burden if sections are restructured. Content alignment is subjective and the existing content may be intentionally different in level of detail between narrative and formal mappings.

**Impact:** No changes.

## Recommendation

**Disposition A: Persuasive — Add Cross-Links and Align Content**

This is a well-founded editorial improvement request. The reporter identifies a genuine usability gap: Section 4's mapping narrative and the per-profile Mappings tabs serve overlapping purposes but are not connected or consistent. The cluster of related tickets from the same ballot round (FHIR-47673, FHIR-47384, FHIR-47377) all point to alignment problems in this IG, reinforcing that this is a systemic issue worth addressing. Adding cross-links is low-cost, and a content alignment audit — even if done incrementally — will improve the IG's reliability as a reference. The work group should consider addressing this ticket alongside FHIR-47673 and FHIR-47384/47377 for efficiency.
