## Description

The submitter reports that the graphical representation of the **Imaging Measurements Container** in one section of the IG does not correspond to the representation shown in **section 4.1**. The request is to align the diagrams so they are consistent throughout the IG.

This is an editorial/consistency issue: diagrams in different sections of the DICOM SR to FHIR Resource Mapping IG depict the Imaging Measurements Container differently, which can confuse readers about the intended structure and mapping scope.

## Proposed Dispositions

### Disposition A: Persuasive (Align Diagrams)

**Action:** Accept the change request and update all graphical representations of the Imaging Measurements Container throughout the IG to use a single, consistent diagram that matches the authoritative definition (likely the one in the overview section). Review sections 3.x and 4.x to ensure every diagram showing the Imaging Measurements Container includes the same components (or explicitly notes any intentional differences).

**Rationale:** Diagram consistency is essential for an IG that maps complex DICOM structures. Inconsistent diagrams create confusion about which components belong to the Imaging Measurements Container, potentially leading to incorrect implementations. This is a straightforward editorial fix with clear benefit and no normative impact.

### Disposition B: Persuasive with Modification (Consolidate with FHIR-47377)

**Action:** Accept the change request, but consolidate the resolution with [FHIR-47377](https://jira.hl7.org/browse/FHIR-47377), which reports the same underlying issue (diagram misalignment for the Imaging Measurements Container) from a different section reference. Perform a comprehensive audit of all Imaging Measurements Container diagrams across the entire IG and align them in a single editorial pass.

**Rationale:** FHIR-47377 and FHIR-47384 describe the same root cause from different vantage points. Resolving them together ensures a thorough fix and avoids piecemeal corrections that might leave other inconsistencies. This approach is more efficient for both editors and reviewers.

### Disposition C: Not Persuasive (Intentional Variation)

**Action:** Reject the change request, noting that different sections may intentionally show different levels of detail or different perspectives of the Imaging Measurements Container depending on the context being discussed.

**Rationale:** Some IGs deliberately simplify or expand diagrams in different sections to focus on the topic at hand. If the variations are intentional, adding a clarifying note to each diagram explaining why it differs from others would address the concern without requiring full alignment.

## Recommendation

**Disposition B: Persuasive with Modification** — This ticket and FHIR-47377 clearly describe the same underlying problem (inconsistent Imaging Measurements Container diagrams across the IG). Consolidating them for a single comprehensive diagram audit is the most efficient resolution. A complete pass through all sections ensures no diagram misalignments remain, improving IG clarity for implementers. The editorial nature of this fix means there is no normative risk, and the benefit to readers is clear.
