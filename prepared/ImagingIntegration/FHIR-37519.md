## Summary

The submitter requests that reserved trademarks—specifically FHIR, HL7, IHE, and SNOMED—be marked with the registered trademark symbol (®) throughout the Radiation Dose Summary for Diagnostic Procedures on FHIR Implementation Guide, consistent with the trademarks identified in the IG's Home page Trademarks section.

A comment on the ticket (JIRAUSER17659, 2022-06-14) clarifies that not every instance of a trademarked term needs the ® symbol; from a regulatory perspective, only a few prominent uses are sufficient. The comment references a Zulip discussion in the `ig-publishing-requirements` stream (topic: "Enforcing (c) symbols") for further context on this approach.

This is a recurring theme across HL7 IGs. Multiple similar tickets have been filed for other specifications (see Related Jira Tickets below), with resolutions ranging from "Persuasive" to "Persuasive with Modification." The general consensus is that trademark symbols should appear at least on first use or in key locations, rather than on every occurrence.

---

## Details

### What Is Requested

Add the ® symbol to all reserved trademarks as identified in the Trademarks section of the IG Home page. The affected terms include FHIR, HL7, IHE, and SNOMED.

### Existing Comment / Clarification

The sole comment notes that blanket marking of every instance is unnecessary from a regulation perspective. Only a select number of prominent uses (e.g., first use on a page, or in the title/header) need the trademark symbol. This aligns with standard trademark usage practice and with how other HL7 IGs have handled the same issue.

### Specification Context

The Radiation Dose Summary for Diagnostic Procedures on FHIR IG references multiple third-party trademarks. The IG Home page already includes a Trademarks section listing these terms, but the body of the IG does not consistently apply the ® symbol.

## Proposed Dispositions

### Disposition A: Persuasive with Modification

**Verdict:** Accept the request with a scoped modification.

Add the ® symbol for FHIR, HL7, IHE, and SNOMED on **first use** within each major section or page of the IG, rather than on every occurrence. This aligns with standard trademark practice, the commenter's regulatory guidance, and the precedent set by analogous tickets (FHIR-27427, FHIR-24502, FHIR-42282).

**Rationale:** The request is valid—trademark attribution is a legitimate requirement. However, marking every instance is unnecessary and harms readability. A "first use" approach satisfies legal requirements while keeping the IG clean. This is consistent with how other HL7 work groups have resolved identical requests.

### Disposition B: Persuasive

**Verdict:** Accept the request as stated.

Add the ® symbol to all reserved trademarks as identified in the Trademarks section, applying it consistently throughout the IG wherever these terms appear.

**Rationale:** This directly fulfills the submitter's request and provides maximum trademark compliance. However, it may result in excessive symbol usage that clutters the text, which is why similar tickets in other IGs were typically modified to "first use" only.

### Disposition C: Not Persuasive

**Verdict:** Decline the request.

The IG already includes a Trademarks section on the Home page that identifies the reserved trademarks. This is sufficient attribution. Adding ® symbols throughout the body text is unnecessary and is not a technical correction—it is a stylistic/legal preference.

**Rationale:** The Trademarks section already serves the purpose of trademark attribution. Declining avoids unnecessary editorial churn. However, this position conflicts with the precedent set by multiple other HL7 IGs that accepted similar requests.

---

## Recommendation

**Disposition A: Persuasive with Modification** is recommended.

This is the most balanced approach and directly aligns with the precedent from analogous tickets across multiple HL7 work groups. The commenter on this ticket has already acknowledged that only a few prominent uses need the ® symbol from a regulatory perspective. Applying ® on first use per page or section satisfies trademark requirements without cluttering the IG. This resolution is consistent with FHIR-27427 (Persuasive), FHIR-24502 (Persuasive with Modification), and FHIR-42282 (Persuasive), all of which addressed the same fundamental issue in other specifications.
