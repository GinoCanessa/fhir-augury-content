## Description

The ticket requests adding the CDC Catheter-Associated Urinary Tract Infection (CAUTI) Standardized Infection Ratio (SIR) as a fully worked example of a risk-adjusted measure to the Quality Measures IG. The source material is a CQL implementation of the CAUTI measure at [cqframework/dqm-content-qicore-2026](https://github.com/cqframework/dqm-content-qicore-2026/blob/main/input/cql/CAUTI.cql).

The resolution description states: *"Provide a fully worked risk-adjusted measure example, based on the Catheter-Associated Urinary Tract Infection Standardized Infection Ratio measure."*

## Proposed Dispositions

### Disposition A: Accept as Persuasive (Recommended)

**Action:** Accept the resolution as stated — add a fully worked CAUTI SIR risk-adjusted measure example to the Quality Measures IG, with a new dedicated section after the scoring type sections.

**Rationale:** The ticket has clear community support from CQI subgroup and workgroup calls (2026-02-25 and 2026-02-27). An implementation commit already exists in the fhir-cqm repository (4c969a6). The CAUTI SIR is a well-understood, real-world CDC measure that demonstrates risk adjustment using ratio and continuous variable scoring. The companion ticket FHIR-51380 (already published) established conformance changes to support mixed-basis ratio measures, using the same CAUTI use case. Adding this example fills a documented gap — the Zulip "Risk adjustments in eCQMs" thread from 2020 shows community questions about risk adjustment that existing guidance did not fully address with worked examples. The material is sourced from established Cooking With CQL content, minimizing editorial risk.

### Disposition B: Accept as Persuasive with Modification

**Action:** Accept but request that the new section also include a brief conceptual overview of risk adjustment (beyond the CAUTI example) — e.g., when to use ratio vs. continuous variable scoring for risk-adjusted measures, and how risk-adjustment variables (supplemental data) relate to the risk model.

**Rationale:** While the CAUTI SIR example is valuable, the subgroup call notes (2026-02-25) specifically mentioned describing "how to use ratio and continuous variable measures in RA measures, using the RA model (linear regression, etc.)." Ensuring the section includes this conceptual framing — not just the example — would better serve implementers encountering risk adjustment for the first time. This aligns with the broader pattern of related tickets (FHIR-44711, FHIR-42898) showing that risk-adjusted measures require multiple facets of IG support.

### Disposition C: Defer for Additional Review

**Action:** Defer to allow the workgroup to review the PR/commit (4c969a6) in detail before formal acceptance.

**Rationale:** The comment from 2026-02-27 states the plan was to "make a PR first and review it in place." While a commit exists, it is unclear whether a formal PR review was completed. Deferring allows the workgroup to verify the content quality, confirm alignment with the Cooking With CQL source material, and ensure the new section integrates cleanly with the existing scoring type sections and the conformance changes from FHIR-51380.

---

## Recommendation

**Disposition A: Accept as Persuasive.** The implementation work is complete (commit exists), the resolution is well-defined, the source material is established, and both the subgroup and CQI workgroup endorsed the approach. The companion conformance changes (FHIR-51380) are already published, making this a natural follow-on addition.
