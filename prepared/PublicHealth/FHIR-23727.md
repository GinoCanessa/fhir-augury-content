## Description

The reporter finds the "Other" tab in the HAI-LTCF (Healthcare Associated Infections – Long Term Care Facilities) IG confusing. The items listed under "Other" are labeled as "other resources," which implies they are FHIR profiles, but they do not appear to be. Opening each link reveals inconsistent content — some appear to be mappings, others look like examples. The reporter requests clarification on what these items actually are and how they should be categorized.

## Analysis

This ticket is a documentation/navigation clarity issue from the STU ballot period (August 2019). The reporter's core concern is that the "Other" tab in the HAI-LTCF IG aggregates heterogeneous content (mappings, examples, Questionnaires, QuestionnaireResponses) under the misleading label "other resources," making it unclear what each item is or how it relates to the IG.

**Key context:**
1. **FHIR-24488** (Persuasive, Published) already addressed part of this problem by removing Questionnaire and QuestionnaireResponse items from the "Other" page, since they were duplicated on the Questionnaire Instances and Examples page.
2. The HAI-LTCF IG has been published through STU 1.1 (April 2023), meaning the IG structure has evolved significantly since the ticket was filed.
3. The ticket remains Triaged and Unresolved despite the related cleanup work, suggesting it may have been partially addressed by FHIR-24488 but never formally dispositioned.

The residual question is whether the remaining items on the "Other" tab (after the FHIR-24488 cleanup) are now appropriately labeled and categorized, or whether further clarification is still needed.

## Proposed Dispositions

### Disposition A: Persuasive (Clarify remaining "Other" tab content)
- **Action:** Review the current "Other" tab in HAI-LTCF and add brief descriptions or categorization labels (e.g., "Mapping," "Example," "Supporting Artifact") to each remaining item so readers understand what they are.
- **Rationale:** The reporter's concern is valid — mixed content under a vague label harms usability. While FHIR-24488 removed some duplicates, the remaining items may still benefit from clearer labeling.

### Disposition B: Persuasive with Modification (Already partially addressed)
- **Action:** Acknowledge that FHIR-24488 addressed the primary source of confusion (duplicate Questionnaires/QRs). Verify the current "Other" tab content is appropriately labeled in the latest published version. If residual issues exist, document them as a separate future enhancement.
- **Rationale:** The IG has undergone significant revision since this 2019 ticket. The most confusing items were already removed via FHIR-24488. The remaining content may now be adequately clear.

### Disposition C: Not Persuasive (Superseded by FHIR-24488)
- **Action:** Close with a note that FHIR-24488 addressed the core confusion by removing duplicated content from the "Other" page, and that subsequent IG publications have reorganized the content.
- **Rationale:** The IG has been published through STU 1.1 with structural improvements. The "Other" tab terminology is standard IG Publisher behavior and is consistent across FHIR IGs.

## Recommendation

**Disposition B: Persuasive with Modification** — The most actionable path is to acknowledge that FHIR-24488 already resolved the worst of the confusion, verify the current state of the "Other" tab in the published STU 1.1, and close this ticket with a note that the issue has been substantially addressed. If any remaining items still lack clarity, they can be tracked as a minor enhancement in a future cycle. This avoids re-opening resolved work while honoring the reporter's valid feedback.
