## Summary

This change request asks that figures in the PDMP FHIR Implementation Guide be improved in two ways:

1. **Notes text should be moved out of figures** — Notes currently embedded as text within figure images should instead appear outside the figure as captions.
2. **Image resolution should be increased** — All images should be higher resolution so that text and diagrams are more readable.

The request addresses both editorial quality and accessibility concerns, as text embedded in images cannot be read by screen readers.

## Proposed Dispositions

### Disposition A: Persuasive
**Resolution:** Accept the request as stated.

**Rationale:** The request is clear and well-scoped. Embedded text in figures is an accessibility barrier (screen readers cannot parse image text), and low-resolution images reduce readability. Multiple similar tickets across other HL7 IGs have been accepted (e.g., FHIR-30342 in MedMorph). The PDMP IG has now been published as STU1, so the prior deferral reason ("IG withdrawn") no longer applies. All figures in the IG should have notes moved to captions, and images should be regenerated at higher resolution.

**Impact:** Editorial changes to all figures in the PDMP IG. Requires regeneration of diagrams at higher resolution and restructuring of figure notes into caption text. No structural or conformance changes.

### Disposition B: Persuasive with Modification
**Resolution:** Accept the request with scoping modifications. Move embedded notes text out of figures into captions where present, and improve image resolution for diagrams containing text. Apply changes as part of the next STU update cycle. Prioritize figures where embedded text creates the most significant readability or accessibility issues.

**Rationale:** The request is valid and aligns with accessibility best practices and HL7 precedent (FHIR-30342 was resolved as Persuasive with Modification for the same concern). However, "all images" is broad — the modification scopes the work to figures that actually contain embedded notes text and diagrams with readability concerns, rather than requiring every image to be replaced. This also aligns with FHIR-29542's recommendation to adopt the new HL7 publishing template, which may address some formatting concerns automatically.

**Impact:** Targeted editorial changes to specific figures in the PDMP IG. Lower effort than Disposition A while addressing the core accessibility and readability concerns.

### Disposition C: Not Persuasive
**Resolution:** The current figures are adequate for the published STU1 edition.

**Rationale:** The PDMP IG STU1 has been published and is in active use. The figures, while not ideal, convey the necessary information. Reworking all figures introduces risk of introducing errors and delays other substantive work. If the new HL7 publishing template (FHIR-29542) is adopted, figure formatting improvements may come as a natural consequence.

**Impact:** No changes. Figures remain as-is in the current publication.

## Recommendation

**Disposition B: Persuasive with Modification** is recommended.

The request identifies a legitimate accessibility and readability concern that has been consistently upheld across other HL7 IGs (notably FHIR-30342). The prior deferral is no longer valid since the PDMP IG is now published as STU1. However, scoping the work to figures with embedded notes text and diagrams with readability issues (rather than blanket "all images") is more practical and aligns with how similar tickets have been resolved. This work should be coordinated with FHIR-29542 (adopting the new HL7 publishing template) to maximize efficiency.
