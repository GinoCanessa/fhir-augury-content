## Description

The submitter identifies a factual inaccuracy in the PDMP IG introduction. PDMPs track controlled substance **dispenses** (i.e., when a pharmacy hands over a medication to a patient), not controlled substance **prescriptions** (i.e., when a prescriber writes an order). This is a critical distinction because the entire purpose of a PDMP is to monitor actual dispensing events, not prescribing activity.

**Existing Wording:**
> In the United States, every state is deploying a Prescription Drug Monitoring Program (PDMP) which track controlled substance prescriptions within the state

**Proposed Wording:**
> In the United States, every state is deploying a Prescription Drug Monitoring Program (PDMP) which track controlled substance dispenses within the state

**Resolution Description:** Narrative updated

## Analysis

This is a straightforward factual correction. PDMPs are universally understood to track **dispensing** events — pharmacies report to state PDMPs when they dispense controlled substances, not when prescribers write prescriptions. The existing wording is inaccurate and could mislead readers about the fundamental purpose and data model of the IG.

Key considerations:
- **FHIR-29467** already fixed a grammar issue ("track" → "tracks") in the **same sentence** and has been Applied. The combined correct text should read: "...which **tracks** controlled substance **dispenses** within the state."
- **FHIR-29454** addresses the same prescriptions-vs-dispenses theme in a different part of the IG.
- The PDMP IG has been published as **STU1** (October 2024), so this fix should be verified against the published version and applied if not already incorporated.
- The resolution description ("Narrative updated") suggests the intent was accepted previously.

---

## Proposed Dispositions

### Disposition A: Persuasive
**Action:** Accept the proposed wording change exactly as submitted — replace "prescriptions" with "dispenses" in the introduction sentence.

**Rationale:** The change is factually correct and aligns with the established understanding of PDMPs. The submitter's proposed wording directly fixes the inaccuracy with no ambiguity.

**Risk:** Low. However, this does not account for the grammar fix in FHIR-29467 (already Applied), which changes "track" to "tracks" in the same sentence. Applying this independently could create a coordination gap.

### Disposition B: Persuasive with Modification
**Action:** Accept the intent of the change but apply it in coordination with FHIR-29467's fix (already Applied), ensuring the sentence reads: *"In the United States, every state is deploying a Prescription Drug Monitoring Program (PDMP) which **tracks** controlled substance **dispenses** within the state."* Verify the fix against the published STU1 text.

**Rationale:** Combining both corrections into the final sentence ensures consistency. Since FHIR-29467 is already Applied, the implementer applying this ticket should ensure the grammar fix is preserved. The resolution description ("Narrative updated") confirms the workgroup's prior intent to accept.

**Risk:** Minimal. Requires checking the STU1 publication to confirm whether this correction was already incorporated during the publication process.

### Disposition C: Not Persuasive
**Action:** Reject the change. Keep "prescriptions" in the existing wording.

**Rationale:** None identified. The existing wording is factually incorrect — PDMPs do not track prescriptions, they track dispenses. There is no policy or technical reason to retain the inaccurate terminology.

**Risk:** High. Leaving factually incorrect terminology in the IG would undermine credibility and could confuse implementers about the data scope.
