The submitter requests that the `military-service-solor-extension` Code System canonical URL be rooted in `terminology.hl7.org` (THO) rather than `hl7.org/fhir`. A UTG ticket should be created to register the Code System in THO.

The resolution description states: *"While specific direction on this is apparently part of FHIR 5 it is not expressly stated in FHIR 4. It was agreed that WGs will consider these votes 'non-persuasive' at this time."*

## Proposed Dispositions

### Option A: Not Persuasive
**Rationale:** The resolution description already indicates that WGs agreed to treat these votes as non-persuasive at the time, because FHIR R4 does not expressly mandate that IG-defined code systems must be rooted in THO. The US Military Service IG has since been published (FHIR-33681 was resolved as Persuasive with Modification, addressing broader URI governance). Re-opening this specific canonical URL change without clear R4 policy backing or active IG development would be premature.

### Option B: Considered for Future Use (Retain Current Resolution)
**Rationale:** Acknowledge the request as valid in principle—FHIR R5 and evolving THO/UTG policy increasingly favor hosting HL7-defined code systems under `terminology.hl7.org`. However, since R4 lacked this mandate and the IG ballot has concluded, defer action until the IG undergoes its next update cycle, at which point the code system URL can be migrated to THO as part of a broader alignment effort.

### Option C: Persuasive with Modification
**Rationale:** Accept the principle that HL7-authored code systems should be registered in THO, and create a UTG ticket to register `military-service-solor-extension` under `terminology.hl7.org`. Apply this as a prospective change for the next published version of the US Military Service IG, aligning with the direction established in FHIR R5 and UTG governance policy.

## Recommendation

**Option B: Considered for Future Use (Retain Current Resolution)**

The existing resolution accurately reflects the state of affairs: the request is directionally correct but was premature for the R4 ballot cycle. FHIR R5 and evolving UTG/TSMG policy now more clearly support hosting HL7 code systems in THO, but the US Military Service IG is not currently undergoing active revision. Retaining the "Considered for Future Use" resolution preserves the intent for action during the next IG update, without requiring immediate changes to a published specification.
