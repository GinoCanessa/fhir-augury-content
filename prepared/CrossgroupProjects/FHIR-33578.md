## Description

Floyd Eisenberg requests a section or profile in the US Military Service IG to delineate specific exposure risks relevant to veterans. The initial IG project discussions indicated that details of specific field exposures (e.g., burn pit exposure causing respiratory risk, loud noise exposure causing hearing risk) would be extremely helpful for managing veterans post-military discharge or while still serving. The submitter could not find reference to the ability to indicate specific exposure risks and asks how they are or will be addressed.

## Analysis

This ticket represents a **real clinical gap** in the US Military Service IG. The IG currently focuses on documenting military service episodes (time served, branch, discharge status) but lacks the ability to record specific environmental or occupational exposure risks that are critical for post-service clinical care. The PACT Act (2022) and increased awareness of burn pit exposure have made this a high-profile issue for veteran healthcare.

The ticket was previously deferred (11-0-1 vote) as beyond STU1 scope, then deliberately re-opened post-publication — signaling workgroup intent to address it in a future version. The closely related FHIR-33353 (Highest priority) covers the same gap with more detail, suggesting these should be coordinated.

Key design considerations include:
- Whether to use **Observation** (exposure documented), **Condition** (exposure-related health risk), or a dedicated **exposure profile/extension**
- Which terminology to use for exposure types (no standard military-specific code system identified yet)
- Relationship to the ODH (Occupational Data for Health) IG patterns

## Proposed Dispositions

### Disposition A: Persuasive — Define Exposure Risk Profiles
**Action:** Accept the request and define new profiles or extensions for documenting specific military exposure risks in the next version of the IG.

- Create an Observation-based "Military Exposure" profile with a value set for exposure types (burn pit, depleted uranium, loud noise, chemical agents, etc.)
- Add a linked Condition profile or extension for exposure-related health risks
- Coordinate with FHIR-33353 to address both tickets together
- Engage Occupational Medicine SMEs (per FHIR-33356) for exposure categorization

**Rationale:** The clinical need is well-established, the workgroup previously committed to considering this for future versions, and community interest (Zulip discussion) confirms ongoing demand. Addressing this would significantly increase the IG's clinical utility.

### Disposition B: Persuasive with Modification — Reference Existing Patterns, Defer Full Profile
**Action:** Accept the request in principle. For the next ballot, add guidance text describing how to represent exposure risks using existing FHIR resources (Observation, Condition) without defining new profiles. Commit to full exposure profiles in a subsequent version.

- Add an informative section describing recommended patterns for exposure documentation
- Include examples using Observation with appropriate LOINC/SNOMED codes where available
- Create a placeholder extension for exposure type to demonstrate intent
- Defer formal profiling to a version where terminology work is complete

**Rationale:** This provides immediate value and guidance to implementers while acknowledging that a robust exposure terminology is not yet available. Reduces the risk of premature profiling with incomplete code systems.

### Disposition C: Deferred — Coordinate with ODH and Terminology Work First
**Action:** Defer action pending coordination with the Occupational Data for Health (ODH) IG and completion of military-specific exposure terminology work.

- Document the requirement as a future-version commitment
- Initiate a cross-IG coordination effort with ODH for occupational exposure patterns
- Engage VA and DoD stakeholders to identify or develop an appropriate exposure code system
- Re-evaluate when terminology foundations are in place

**Rationale:** Without a standardized terminology for military exposures, any profile defined now may need significant rework. Coordinating with ODH avoids duplicate effort and ensures alignment with civilian occupational health patterns.

---

## Recommendation

**Disposition A: Persuasive — Define Exposure Risk Profiles**

This ticket has already been deferred once and deliberately re-opened, signaling clear workgroup intent to act. The clinical need is well-documented, reinforced by the companion ticket FHIR-33353 (Highest priority) and recent Zulip discussion showing implementers hitting this gap in practice. The PACT Act has increased urgency for exposure documentation in veteran health records. While terminology gaps exist, the IG can define an extensible binding that accommodates future code system development. Coordinating with FHIR-33353 allows both tickets to be resolved together efficiently.
