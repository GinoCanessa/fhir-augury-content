## Description

This ticket requests the addition of a qualification slice on `Practitioner.qualification` in the AU Base Practitioner profile to support Australian Specialist Medical College fellowships (e.g., FRACP, FRCPA).

The original request (from GitHub issue [hl7au/au-fhir-base#463](https://github.com/hl7au/au-fhir-base/issues/463), raised by **davidmckillop** on Aug 7, 2020) identifies that:

- There is no published CodeSystem or ValueSet for qualifications from the Australian Specialist Medical Colleges.
- The [Australian Medical Council (AMC)](https://www.amc.org.au/accredited-organisations/assessment-and-accreditation-of-specialist-medical-programs/specialist-medical-colleges/) website maintains a list of Specialist Colleges.
- Fellowship qualifications issued by each college are stable (e.g., FRCPA from RCPA, FRACP from RACP).
- The request is for:
  - **(a)** A definitive list of Australian Colleges → to populate `Practitioner.qualification.issuer`
  - **(b)** A definitive list of fellowship abbreviations (e.g., FRACP, FRCPA) → to populate `Practitioner.qualification.code`

## Proposed Dispositions

### Disposition A: Applied — Add Fellowship Qualification Slice with New CodeSystem
**Action:** Accept the change request and incorporate the draft work from the `cr-practitioner-qualification` branch. This adds:
- A new CodeSystem defining Australian Specialist Medical College fellowship codes (e.g., FRACP, FRCPA) sourced from the AMC list.
- An extended ValueSet (`au-v2-0360-extended`) bound to `Practitioner.qualification.code`.
- A qualification slice on AU Base Practitioner for college fellowships.

**Rationale:** This has been discussed and agreed by the PA WG since March 2021. Draft implementation already exists. Fellowship qualifications are stable and well-defined. This fills a real gap in the AU Base profile for representing specialist medical practitioner qualifications.

**Risks:** Requires ongoing maintenance of the college list if AMC updates their accredited organisations. The CodeSystem source (AMC website) is not a formal terminology authority.

### Disposition B: Deferred — Wait for Formal Terminology
**Action:** Defer until a formal Australian terminology authority publishes an official CodeSystem for specialist medical college fellowships.

**Rationale:** The AMC website is not a terminology publisher. Using a web page as the authoritative source for a CodeSystem may create maintenance and governance challenges. Deferring until NCTS or another authority publishes these codes would provide a more sustainable solution.

**Risks:** This has been pending since 2020; further deferral may frustrate implementers who need these codes now. There is no indication a formal terminology is forthcoming.

### Disposition C: Consider for Future Use — Approve Concept, Defer Implementation
**Action:** Acknowledge the need and approve the concept, but defer implementation to a future ballot cycle to allow time for broader community review of the draft CodeSystem content.

**Rationale:** The July 2024 comment suggested "move to Consider for Future Use." While the draft branch exists, the new CodeSystem may benefit from broader review to ensure completeness and accuracy of college/fellowship mappings before formal inclusion.

**Risks:** The draft work may go stale. The implementer (Brett Esler) has already done significant work that could be lost or need rework.

## Recommendation

**Disposition A: Applied — Add Fellowship Qualification Slice with New CodeSystem**

This is recommended because:
1. The change has been discussed and formally agreed by the PA Work Group since March 2021.
2. A working draft implementation already exists on a branch with a new CodeSystem and extended ValueSet.
3. The request addresses a genuine gap — there is currently no standard way to represent Australian specialist medical college fellowships in AU Base Practitioner.
4. Fellowship qualifications from specialist colleges are stable and well-understood.
5. Both related tickets (FHIR-46327 and FHIR-46320) have been retracted, leaving this as the primary vehicle for this enhancement.
6. The work has been allocated and actively progressed by the reporter, demonstrating community commitment.

The workgroup should review the draft branch content for completeness and accuracy, then apply the change.
