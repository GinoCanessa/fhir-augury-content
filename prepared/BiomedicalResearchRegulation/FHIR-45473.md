## Description

The request is to correct the display value for NCI Thesaurus code **C133936** to use the PQ/CMC preferred term (PT): **"Storage at ~20 degrees C (+/−) 5 degrees C."** The current display text does not match the PQ/CMC preferred terminology as listed on the NCIt EVS site.

## Proposed Dispositions

### Disposition A: Persuasive

**Rationale:** The request is straightforward and consistent with precedent. Sibling tickets FHIR-45471 and FHIR-45472, which made the same kind of correction (aligning display values with PQ/CMC preferred terms from NCIt EVS), were both resolved as **Persuasive**. Code C133936 should use the standard PQ/CMC preferred term for its display text. The only reason this was previously deferred was that storage conditions were out-of-scope for STU1 and STU2 — not because the change itself was contested. Now that the ticket has been re-opened post-STU2, and Stage 3 stability work is in scope, the correction should be applied.

**Action:** Update the display value for C133936 to the PQ/CMC preferred term: "Storage at ~20 degrees C (+/−) 5 degrees C."

### Disposition B: Considered for Future Use

**Rationale:** If Stage 3 stability profiles are still not ready for ballot, the work group may choose to defer once more. Storage conditions terminology is tied to the stability study profiles, and applying this change in isolation without the broader stability content may be premature.

**Action:** Defer to a future ballot cycle when Stage 3 Stability profiles are actively being balloted. Document the deferral rationale.

### Disposition C: Persuasive with Modification

**Rationale:** Accept the intent of the change but verify the exact preferred term text against the current NCIt EVS listing for C133936 before applying, as preferred terms can be updated by NCI over time. The display value should match the current NCIt EVS PT at the time of publication.

**Action:** Confirm the current NCIt EVS preferred term for C133936 and update the display value accordingly.
