## Description

The submitter (Genevieve Luensman) requests adding a work-related use case to the Retrieval of Real World Data for Clinical Research (RWD) IG. The ticket explicitly references [FHIR-40098](https://jira.hl7.org/browse/FHIR-40098), which requests including Occupational Data for Health (ODH) in section 3 of the IG. ODH is a framework of information about a person's work (see [hl7.org/fhir/us/odh](http://hl7.org/fhir/us/odh/)) that is not formally in IPS but is considered important for clinical research.

## Proposed Dispositions

### Disposition A: Persuasive

**Action:** Accept the request and add a work-related use case to the RWD IG.

**Rationale:** The ticket was originally resolved as "Considered for Future Use" and then deliberately re-opened post STU publication, signaling intent to address it in the next iteration. ODH has since gained acceptance in other IGs (QI Core accepted FHIR-41286 and FHIR-41287 as Persuasive with Modification), demonstrating growing community support. Work-related exposures (occupational hazards, chemical exposures, work environment factors) are clinically significant for real-world data research. The companion ticket FHIR-40098 for including ODH in section 3 provides the data framework that this use case would leverage.

**Impact:** The RWD IG would need a new use case section describing work-related research scenarios (e.g., occupational exposure studies, work-environment health outcomes). This would require coordination with the ODH IG and would need to define which ODH profiles/resources are relevant. Moderate editorial and technical effort.

### Disposition B: Persuasive with Modification

**Action:** Accept in principle but scope to a single, well-defined work-related use case (e.g., occupational exposure tracking) rather than broad coverage.

**Rationale:** While the need for work-related data in clinical research is valid, the RWD IG is still maturing. Starting with one concrete, well-scoped use case (such as tracking occupational exposures that lead to health conditions) would be more manageable. This aligns with the original ballot resolution's acknowledgment that future iterations should incorporate work-related content incrementally. The modification limits initial scope while establishing the pattern for expansion.

**Impact:** Requires adding one use case section and referencing relevant ODH profiles. Lower effort than full coverage. Should be paired with resolution of FHIR-40098 (ODH in section 3) to ensure the data framework is present.

### Disposition C: Not Persuasive

**Action:** Decline the request; keep work-related use cases out of the current RWD IG scope.

**Rationale:** The original 7-0-0 vote acknowledged this was not needed for the current guide iteration. While the ticket was re-opened post-publication, no specific studies or implementer demand for work-related data retrieval in the RWD context have been documented. The RWD IG has other open items (FHIR-39905, FHIR-39960) that may be higher priority. Adding use cases without concrete implementer requirements could dilute the IG's focus.

**Impact:** No changes required. The request could be re-raised when a concrete use case with implementer backing is available.

---

## Recommendation

**Disposition B: Persuasive with Modification** is recommended. The re-opening of this ticket post-STU publication and the growing acceptance of ODH in other IGs (QI Core) demonstrate legitimate momentum for including work-related data in clinical research contexts. However, starting with a single, well-defined occupational health use case (rather than broad coverage) would be more practical given the IG's maturity level. This disposition should be coordinated with FHIR-40098 to ensure ODH profiles are included in the IG's data framework. Amanda Deering and Lori Fourquet (per the reporter's December 2023 comment) should be engaged in scoping the use case.
