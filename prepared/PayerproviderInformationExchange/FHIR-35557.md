## Analysis

### Context

This ticket is part of a systematic effort to move HL7-defined CodeSystem and ValueSet canonical URLs from `hl7.org/fhir` to `terminology.hl7.org` (THO). The HL7 Unified Terminology Governance (UTG) process requires that terminology artifacts be registered in THO with canonical URLs rooted under `terminology.hl7.org`, rather than under the FHIR specification domain.

This specific ticket (Comment 7) requests re-anchoring a **Code System** used in the CARIN Digital Insurance Card IG. It is one of at least five sibling tickets (FHIR-35552 through FHIR-35557) targeting different CodeSystems and ValueSets in the same IG, and part of a broader wave of similar requests across CARIN-family IGs (C4BB, etc.).

### History

The ticket has been resolved as "Considered for Future Use" **twice** (March 2022 and August 2022, both with 11-0-1 votes), but each time was re-opened after the subsequent STU publication cycle. The WG acknowledged the need but deferred action pending the Vocabulary WG formalizing the process for IG-originated terminology registration in THO. As of April 2024, the ticket remains Triaged and Unresolved.

### Key Considerations

1. **Cross-WG dependency**: Requires coordination with the Vocabulary WG and UTG process. Earlier deferrals cited the Vocabulary WG not having formalized the process.
2. **Batch resolution opportunity**: This ticket should be considered together with its siblings (FHIR-35552–35556) for a coordinated resolution.
3. **Breaking change potential**: Changing canonical URLs is a breaking change for implementations that reference the current URLs. A redirect/alias strategy or versioned transition may be needed.
4. **Maturity**: The CARIN DIC IG has gone through STU1 and STU1.1 publication since this ticket was filed; a future STU2 ballot cycle could be an appropriate vehicle for this change.

## Proposed Dispositions

### Disposition A: Applied — Coordinate UTG Registration

**Resolution:** Applied (with coordination)

Move the affected Code System canonical URL to `terminology.hl7.org` and create a corresponding UTG ticket to formally register the Code System in THO. Coordinate with the Vocabulary WG to ensure the UTG submission is properly processed. Apply as part of the next CARIN DIC ballot cycle (STU2), and process all sibling tickets (FHIR-35552–35556) together to ensure consistency.

**Rationale:** This is the correct long-term approach per HL7 terminology governance policy. The Vocabulary WG has had significant time since 2022 to formalize the process, and it is reasonable to move forward now. Batching all sibling tickets reduces coordination overhead.

### Disposition B: Deferred — Await Vocabulary WG Process Formalization

**Resolution:** Deferred

Defer this ticket until the Vocabulary WG has published formal guidance on how IG-originated CodeSystems and ValueSets should be migrated to THO, including canonical URL transition strategies and UTG submission procedures. Re-evaluate at the next CARIN DIC ballot cycle.

**Rationale:** Previous attempts to resolve this ticket were reverted because the Vocabulary WG process was not yet formalized. Acting without a clear migration path risks inconsistency across IGs and potential implementation breakage. However, this approach risks indefinite deferral given the ticket has already been deferred twice since 2022.

### Disposition C: Not Persuasive — Retain Current URLs

**Resolution:** Not Persuasive

Retain the existing canonical URL under `hl7.org/fhir` for the Code System. The current URL is already in use by implementations, and changing it introduces a breaking change with limited practical benefit if the Code System is well-understood and stable.

**Rationale:** While THO anchoring is the stated HL7 policy direction, the practical impact of migrating an established canonical URL may outweigh the governance benefit, especially for a mature IG. This disposition would need strong justification given the 11-0-1 votes acknowledging the validity of the request.

---

## Recommendation

**Disposition A: Applied — Coordinate UTG Registration**

This ticket has been open since January 2022 and deferred twice. The underlying request aligns with HL7 terminology governance policy, and the workgroup has consistently voted (11-0-1) that the change is valid. With the CARIN DIC IG now past STU1.1, the next ballot cycle (STU2) provides a natural vehicle for this breaking change. The workgroup should coordinate with the Vocabulary WG to batch-process all sibling tickets (FHIR-35552 through FHIR-35557) together, creating corresponding UTG tickets and applying the canonical URL migration in a single coordinated effort.
