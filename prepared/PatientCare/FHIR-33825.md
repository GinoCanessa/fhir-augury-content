## Summary

This ticket requests that the **body-site-eye** Value Set canonical URL be moved from the `hl7.org/fhir` namespace to `terminology.hl7.org` (THO — the HL7 Terminology server). The submitter notes that a UTG (Unified Terminology Governance) ticket should be created to register the Value Set in THO. This is one of a series of similar tickets filed by Ron G. Parker requesting canonical URL migration for various Ophthalmology Retinal Value Sets.

## Details

### Description

The body-site-eye Value Set canonical URL should be rooted in `terminology.hl7.org` (THO), not rooted in `hl7.org/fhir`. A UTG ticket should be created to register the Value Set in THO.

## Proposed Dispositions

### Disposition A: Persuasive (Accept)

**Rationale:** The request is technically sound. HL7 terminology governance policy directs that Value Sets managed under UTG should have canonical URLs rooted in `terminology.hl7.org`, not `hl7.org/fhir`. The sibling ticket FHIR-33763 (Devices WG) was already resolved as Persuasive and Applied, establishing precedent. Accepting this ticket aligns the Ophthalmology Retinal IG's body-site-eye Value Set with current THO practices.

**Action Required:**
1. Create a UTG ticket to register the body-site-eye Value Set in THO with a `terminology.hl7.org`-rooted canonical URL.
2. Update the Ophthalmology Retinal IG to reference the new canonical URL.
3. Coordinate with the batch of related tickets (FHIR-33822 through FHIR-33834) to handle all Ophthalmology Retinal Value Set URL migrations together.

### Disposition B: Persuasive with Modification

**Rationale:** Accept the intent but defer actual migration until the entire batch of related Ophthalmology Retinal canonical URL tickets (FHIR-33822–33834) can be processed as a coordinated set. This avoids piecemeal changes and ensures consistency across all Value Sets in the IG.

**Action Required:**
1. Accept as Persuasive with Modification, noting that implementation should be bundled with all sibling canonical URL migration tickets for this IG.
2. Assign a single UTG submission covering all affected Value Sets.
3. Schedule the coordinated migration for a future IG publication cycle.

### Disposition C: Not Persuasive (Decline)

**Rationale:** The Value Set is part of the Ophthalmology Retinal IG which has not been widely adopted, and the existing canonical URL under `hl7.org/fhir` is functional. Changing canonical URLs carries a breaking-change risk for any implementers already referencing the current URL. The work group may choose to defer this until the IG reaches a more mature ballot stage where such a change is less disruptive.

**Action Required:** No action; document rationale that URL migration is deferred pending broader IG maturity and adoption review.

## Recommendation

**Disposition A: Persuasive (Accept)**

The request aligns with HL7's established terminology governance policy for THO-managed Value Sets. The identical pattern has already been accepted and applied in the Devices WG (FHIR-33763), making this a straightforward precedent-following decision. The body-site-eye Value Set canonical URL should be migrated to `terminology.hl7.org` via a UTG ticket. Ideally, the work group should coordinate this with the full batch of related Ophthalmology Retinal tickets to process them efficiently together.
