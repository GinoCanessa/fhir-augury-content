## Description

At the bottom of the Artifact Overview page (`fhir_artifacts_overview.md`) the text says **"The artifacts are of four types:"** but five types are listed below it:

1. Profiles
2. Extensions
3. Code Systems
4. Value Sets
5. Operations

The resolution is to change "four" to "five" to match the actual list.

## Resolution Description

> Will updated the number of artifacts

## Feature Proposal

Change the text from "The artifacts are of four types" to "The artifacts are of five types" on the FHIR Artifacts Overview page of the Post-Acute Orders IG.
