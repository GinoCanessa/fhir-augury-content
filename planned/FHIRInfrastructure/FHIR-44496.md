## Summary

The element definition for `ExampleScenario.process.title` has a grammatical error — the word "of" is missing between "label" and "the process." The current definition reads:

> A short descriptive label the process to be used in tables or diagrams.

The approved fix adds the missing preposition:

> A short descriptive label **of** the process to be used in tables or diagrams.

## Cross-References

No cross-references found for this ticket in Zulip, GitHub, or Confluence. No existing GitHub issues or pull requests reference FHIR-44496.

## Feature Proposal

Insert the word "of" into the `<definition>` value for `ExampleScenario.process.title` in the ExampleScenario StructureDefinition source file.
