## Problem Statement

`Evidence.studyDesign` currently has an Extensible binding to the locally-defined ValueSet at `http://hl7.org/fhir/ValueSet/study-design` (file `source/evidence/valueset-study-design.xml`). This ValueSet has been migrated to the HL7 Terminology (THO) server at `http://terminology.hl7.org/ValueSet/study-design`, but the binding in the FHIR core spec was not updated in time for the R6 ballot. The binding reference and the local ValueSet copy must be updated accordingly.

## Resolution Description

> We will update the studyDesign binding to reference the ValueSet which is now hosted on THO.

## Detailed Implementation Plan

### Step 1: Update the Evidence StructureDefinition Binding

**File:** `source/evidence/structuredefinition-Evidence.xml`

```xml

<!-- After -->

```

**File:** `source/evidence/valueset-study-design.xml`  
**Action:** Delete the entire file (1,891 lines). The canonical content now lives at THO (`http://terminology.hl7.org/ValueSet/study-design`).
