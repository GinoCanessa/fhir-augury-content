## Summary

This ticket is a Technical Correction against the **US Breast Cancer Radiology
Reporting (FHIR)** IG. On the page directed at "Clinicians and cancer domain
experts" (under a "Where to Focus" heading), two links labelled
"Modeling approach" and "logical models" both currently target the
**Breast Radiology Report** section. The reporter asks that those two link
targets be corrected so that "Modeling approach" links to the modeling
approach section and "logical models" links to the logical models section,
respectively. The ticket has been resolved as **Persuasive** with a commitment
to improve the link targets and surrounding link descriptions in the next
ballot. No cross-referenced Jira tickets, Zulip discussions, or GitHub items
were surfaced for this ticket.

## Proposed Dispositions

### Disposition A: Accept As Requested

#### Proposal

Update the "Where to Focus: Clinicians and cancer domain experts" entry on
the IG's audience-guidance page so that:

- The link text "Modeling approach" points to the IG's **Modeling Approach**
  page/section (not the Breast Radiology Report section).
- The link text "logical models" points to the IG's **Logical Models**
  page/section (not the Breast Radiology Report section).

In addition, refresh the surrounding sentence so the link descriptions
accurately reflect the new targets (e.g., "Review the *modeling approach*
used to derive the logical models, then explore the *logical models*
themselves."). No structural changes to the IG, no resource or profile
changes — narrative/page-link change only.

#### Justification

This is exactly what the reporter asked for and matches the recorded
resolution ("Persuasive — we agree, and will improve the links and
description of the links in the next ballot"). The current targets are
demonstrably wrong (both labels point to the same Breast Radiology Report
section), which actively misleads the "clinicians and cancer domain experts"
audience the page is meant to orient. Fixing the anchors and clarifying the
prose has zero impact on conformance, profiles, or downstream implementers,
and aligns with the IG editorial principle that audience-routing pages should
deep-link into the most relevant content for that audience.

---

### Disposition B: Alternative Approach

#### Proposal

Rather than only retargeting two anchors, restructure the "Where to Focus:
Clinicians and cancer domain experts" block into a short itemized reading
path:

1. **Modeling Approach** — link to the Modeling Approach page.
2. **Logical Models** — link to the Logical Models index, with a one-line
   description of what the clinician will find there.
3. **Breast Radiology Report** — keep a link to the Breast Radiology Report
   section as the concrete example that ties the first two together.

Apply the same itemized treatment to the other "Where to Focus" audience
blocks on the page so the navigation pattern is consistent across audiences.

#### Justification

The reporter's underlying need is that a clinician landing on the audience
page can find the right starting content. Disposition A fixes the broken
anchors but leaves the underlying pattern (run-on sentence with embedded
links pointing to overlapping content) intact, which is what allowed the
two links to drift onto the same target in the first place. Promoting the
links to a short numbered reading path makes the intent explicit and is
easier to keep correct on subsequent edits. The trade-off is a slightly
larger editorial change than the ticket strictly requests, and it touches
sibling audience blocks rather than only the clinician block.

---

### Disposition C: Decline

#### Proposal

Decline on the grounds that the affected text is purely orientational
narrative on a "Where to Focus" page and does not affect any normative
content, profile, value set, or implementation behavior; readers can still
reach the Modeling Approach and Logical Models pages via the IG's primary
navigation/table of contents.

#### Justification

A defense for declining would lean on scope discipline — the content is
non-normative, the target pages are reachable through other navigation, and
ballot cycles are expensive. However, this rationale is weak here: the links
are demonstrably incorrect (both labels resolve to the same unrelated
section), the fix is editorial and low-risk, and the work group has already
agreed (Persuasive) to make the change. Declining would contradict that
recorded resolution.

---

### Recommendation

**Recommended disposition:** A

The ticket has already been resolved as **Persuasive** with an explicit
commitment to fix the link targets and descriptions in the next ballot, so
Disposition A is the literal execution of that commitment. The change is
narrative-only, has no normative impact, and addresses a clear defect (two
labels pointing at the same wrong section). Disposition B is attractive as a
follow-on editorial improvement, but it expands scope beyond what the
reporter and the resolution agreed to; if the IG editor wants to apply the
itemized "reading path" pattern, that should be tracked as a separate
editorial ticket. Disposition C is not defensible given the recorded
resolution.
