## Summary

The reporter, commenting on the US Occupational Data for Health (ODH) FHIR
Implementation Guide (issue #18 in the ODH ballot/comment cycle), questions
the IG's framing of itself as US-focused yet "intended to support
international needs." Specifically, the reporter asks whether the design
team has considered the international dimension of paid/personal time off
(PTO) — noting that PTO entitlement and usage vary widely between the US
and other jurisdictions (e.g., European countries) — and how that
variation interacts with the IG's stated international intent for value
sets covering Work Schedule, Employment Status, and Work Classification.

The ticket was originally dispositioned by the Public Health Work Group
(PHWG) on 2023-01-16 as **Considered for Future Use** (vote 20-0-6,
moved by Craig Newman / Genny Luensman), with the rationale that PTO
"is not part of the considerations for the current model." That
resolution was subsequently reverted by Lynn Laakso on 2023-03-22 and
the ticket was re-opened post-ODH 1.3 publication; it currently sits in
**Triaged** state with no assignee.

No related Jira tickets, GitHub items, or Zulip discussions were
surfaced via cross-reference or search; the only linked artifact is the
PHWG WGM minutes page on Confluence cited in the approval comment.

## Proposed Dispositions

### Disposition A: Accept As Requested

#### Proposal

Treat the ticket literally as a request to add Personal Time Off (PTO)
modeling to the ODH IG so the international framing is substantively
reflected. Concretely, in the US ODH FHIR IG (Public Health WG):

- Add a new ODH observation profile (e.g., `odh-PaidTimeOff`) layered on
  `Observation`, capturing PTO entitlement (e.g., annual hours/days) and
  PTO usage (used/remaining for a stated period).
- Define or bind a small value set for PTO category (e.g., vacation,
  sick, parental, statutory holiday) that is structured to admit
  jurisdiction-specific extensions.
- Reference the new profile from the existing Employment Status / Work
  Schedule narrative in the IG, and update the "US vs International"
  paragraph to describe how PTO is represented.

#### Justification

This directly addresses the reporter's concern that the IG advertises
international applicability for the time-related value sets but has no
representation for a clinically and occupationally significant
variable (PTO) that varies dramatically by jurisdiction. Adding the
profile keeps ODH's existing Work Schedule / Employment Status framing
intact and extends it in the same shape (Observation-based) used
elsewhere in the IG, so implementers see a consistent pattern.

---

### Disposition B: Alternative Approach

#### Proposal

Do not add new PTO data elements to the ODH model. Instead, **clarify
the IG's "US versus International Perspective" narrative** so that:

1. The boundary of ODH's international intent is stated explicitly: the
   IG asserts that Work Schedule, Employment Status, and Work
   Classification value sets are *concept-compatible* with international
   use, but the IG does not attempt to model jurisdiction-specific
   leave/PTO entitlements.
2. Implementers needing PTO are pointed to extension mechanisms (FHIR
   extensions on Observation/Patient or jurisdictional profiles) rather
   than a core ODH change.
3. A "Future Considerations" subsection lists PTO and similar
   jurisdiction-sensitive concepts as candidates if a clear cross-realm
   use case emerges.

#### Justification

The reporter's question is fundamentally about scope clarity, not a
demonstrated implementer demand for PTO observations within ODH's
occupational-health use cases (exposure, return-to-work, surveillance).
Tightening the IG narrative resolves the "US vs International"
inconsistency the reporter flagged without committing the WG to
modeling a domain (leave entitlements) that is largely outside ODH's
clinical and public-health surveillance focus and overlaps with
HR/workforce systems. This is also the lightest-touch resolution
compatible with the prior PHWG vote that PTO "is not part of the
considerations for the current model."

---

### Disposition C: Decline

#### Proposal

Disposition: **Not Persuasive — out of scope for ODH.** Close the
ticket with the rationale that PTO entitlement and usage are
HR/workforce-management data, not occupational health data, and are
therefore outside ODH's scope. The IG's existing Work Schedule,
Employment Status, and Work Classification value sets remain framed as
conceptually international, but PTO modeling is not in scope for this
IG and the WG declines to take it on.

#### Justification

This restores, in substance, the original 2023-01-16 PHWG vote (20-0-6,
"Considered for Future Use … not part of the considerations for the
current model") which was procedurally reverted on 2023-03-22 to allow
re-triage post-1.3, not because the WG had changed its mind on the
merits. PTO is reasonably modeled by HR systems and, where a clinical
care plan needs to know about it, by FHIR extensions or local profiles
on `Observation` / `Patient`. There is no Zulip or GitHub activity
suggesting an implementer community is blocked on PTO in ODH, and no
linked tickets indicate a broader pattern of requests. Declining keeps
ODH's scope tight and avoids encoding a jurisdiction-sensitive concept
into a US-realm IG.

---

### Recommendation

**Recommended disposition:** B

The reporter's underlying complaint is a documentation/scope
inconsistency: the IG simultaneously claims international intent for
half of its time-related value sets and offers no model for an obvious
international-variation concept (PTO). Disposition C is defensible on
scope grounds and aligns with the original PHWG vote, but it leaves the
narrative inconsistency the reporter pointed at unaddressed and risks
the same comment recurring in the next ballot cycle. Disposition A
solves the inconsistency by adding a profile, but commits the WG to
modeling leave/entitlement data with no surfaced implementer demand,
no Zulip discussion, and no linked tickets — a significant scope
expansion for a single comment ticket that is now seven years old.

Disposition B threads the needle: it tightens the IG's "US versus
International Perspective" wording so the international claim is
accurate and bounded, points implementers at extension mechanisms for
PTO, and explicitly parks PTO as a future-consideration item. That
honors the reporter's concern, preserves the prior WG vote's substance,
and costs only a narrative edit in the ODH IG. The PHWG should weigh in
on whether the future-considerations note should also enumerate any
adjacent jurisdiction-sensitive concepts (statutory holidays, parental
leave categories, etc.) that have surfaced in other comments.
