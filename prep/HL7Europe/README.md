#### Topic: Manifest bundle and imaging endpoint semantics

These tickets all interrogate how the EU Imaging Study Report manifest exposes images for retrieval — what bundle type the manifest is, which endpoints (IID, WADO-RS, WADO-URI) are required or optional, when KOS attachments are needed, how access works across NCPs, and what should happen when no endpoint is available. They share the same downstream impact on the `Manifest-ImImagingStudy` profile and the imaging `Endpoint` / `DocumentReference` obligations.

##### Linked Ticket Group: FHIR-51273

FHIR-51339 explicitly links to FHIR-51273: both ask the IG to clarify how a `DocumentReference.content.attachment.url` resolves to an image when crossing networks behind National Contact Points, so they must be resolved together to give a single coherent answer about cross-border image URLs.
