#### Topic: CQFM Measure-profile slice short-description revisions

All four tickets are from the same reporter and ask the work group to revise the `short` descriptions on sliced elements of CQFM Measure profiles in the Quality Measures IG. They cover the Publishable Measure (`Measure.relatedArtifact` predecessor/citation slices, §17.53.1.1), the Ratio Measure (`Measure.group.population` slices, §17.54.1.1), and the Continuous Variable Measure (`Measure.group.population` slices, §17.48.1.1). FHIR-45682 is a clone of FHIR-45681 against the CQFM IG; the others address sibling profiles with the same underlying clarity problem, so they are best dispositioned as one coordinated profile-documentation pass.
