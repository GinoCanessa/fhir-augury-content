## Summary

The reporter notes that a figure in the CPG-on-FHIR (Clinical Guidelines)
implementation guide labels its caption with a "1" footnote marker, but
the corresponding reference #1 is missing from the citation list under
the diagram (only reference #2 currently appears). The ticket asks that
the missing citation be added — specifically Tailor et al., "An
Evaluation Framework for a Novel Process to Codevelop Written and
Computable Guidelines," *American Journal of Medical Quality* 38(5S)
S35–S45, September/October 2023, DOI 10.1097/JMQ.0000000000000140 — so
that readers understand the diagram originates from that evaluation
manuscript rather than from the Aziz et al. paper currently cited as
reference #2. The reporter also observes that reference #2 (Boxwala /
Aziz) could remain but is largely redundant, since CPG-on-FHIR already
discusses the Boxwala rubric in another section. No GitHub PRs, Zulip
threads, or formally linked Jira tickets are recorded for this issue;
the cross-reference index returns zero hits, so the request currently
stands on its own as an editorial correction.

## Proposed Dispositions

### Disposition A: Accept As Requested

#### Proposal

Persuasive. In the CPG-on-FHIR (Clinical Guidelines) IG, locate the
figure whose caption ends with the footnote marker "1" (above the
existing reference #2 in the same section), and add the missing
reference #1 immediately above the existing #2 entry, formatted to
match the IG's existing reference style:

> 1. Tailor A, Robinson SJ, Matson-Koffman DM, Michaels M, Burton MM,
>    Lubin IM; on behalf of the Workgroup on Evaluation of the
>    Integrated Process. An Evaluation Framework for a Novel Process to
>    Codevelop Written and Computable Guidelines. *American Journal of
>    Medical Quality.* 2023;38(5S):S35–S45.
>    doi:10.1097/JMQ.0000000000000140.

Leave the existing reference #2 (Boxwala / Aziz) in place. No diagram
asset itself needs to change; this is purely a narrative/citation fix
in the page that hosts the figure.

#### Justification

The reporter is the CPG-on-FHIR lead and is identifying a literal
attribution gap: the figure carries a "1" footnote marker for which no
reference exists, which currently mis-attributes the diagram to the
Aziz paper. Adding the missing citation is the smallest possible change
that resolves the defect, restores correct attribution to the Tailor et
al. evaluation manuscript that the diagram is actually drawn from, and
preserves the existing reading order. No FHIR conformance, terminology,
or resource semantics are affected — this is exactly the kind of
editorial correction that the "Technical Correction" issue type is
intended for.

---

### Disposition B: Alternative Approach

#### Proposal

Persuasive with Modification. Add the Tailor et al. citation as
reference #1 (per Disposition A) **and** also remove or demote
reference #2 (Boxwala / Aziz) under the same figure, since the
reporter notes that the Boxwala rubric is already discussed and cited
in another section of CPG-on-FHIR (the levels-of-knowledge section) and
the newly added Tailor manuscript itself discusses the Boxwala rubric.
Replace the inline #2 with a one-sentence "see also" cross-link to the
existing levels-of-knowledge section so that the figure caption has
exactly one footnote — the one that actually appears in the figure
label.

#### Justification

This addresses the reporter's secondary observation that #2 is
"less necessary" and is what creates the visual confusion (a figure
captioned "1" sitting above a list whose only entry is "2"). Aligning
the footnote markers in the caption with the entries beneath it is a
stronger fix than merely adding the missing entry, and consolidating
the Boxwala citation in a single place reduces duplication across the
IG. The cost is slightly more editorial review than Disposition A
(removing a reference touches a second editor's work), but it leaves
the section materially cleaner.

---

### Disposition C: Decline

#### Proposal

Not Persuasive. Close the ticket without changes on the basis that the
diagram's intended attribution is conveyed elsewhere in the IG and the
missing reference number is a minor cosmetic issue that does not affect
the normative content or implementer behaviour.

#### Justification

This would be defensible only if the diagram were already correctly
attributed in surrounding prose and the missing "1" marker were a stray
artifact rather than a real citation. The ticket's text contradicts
that reading: the reporter — who is the editor familiar with the
diagram's provenance — explicitly identifies that the figure currently
appears to be sourced from Aziz when it is in fact from Tailor et al.
Declining would leave a known mis-attribution in a published HL7
artifact, with potential downstream implications for how the diagram is
cited or reused. The rationale for declining is therefore weak.

---

### Recommendation

**Recommended disposition:** B

Disposition B is recommended. Disposition A is the strict literal
reading of the request and is fine on its own, but the reporter's own
description goes a step further: they note that reference #2 is
redundant with the levels-of-knowledge section of CPG-on-FHIR and that
the Tailor manuscript itself covers the Boxwala rubric. Acting on both
points in a single edit avoids leaving an obviously-mismatched caption
("1" in the figure, only "2" listed below) that would likely produce a
follow-up ticket, and consolidates the Boxwala citation in the place
the IG already discusses it. The change is purely editorial, scoped to
one section, and entirely within the CDS work group's authority over
the Clinical Guidelines IG. Disposition C should not be adopted — the
reporter has identified a genuine attribution defect that the work
group should fix.
