### Change Request

#### Topic: PQ-CMC terminology and value-set corrections

Both tickets are Block-Vote-3 PQ-CMC change requests dealing with how the IG
manages and presents terminology drawn from NCI Thesaurus. FHIR-45473 is a
display-text correction for NCI Thesaurus code C133936 (storage condition);
FHIR-45834 asks the work group to anchor the IG's Route of Administration
value set in the HL7 Terminology service (THO) so it can be reused beyond
PQ-CMC. Together they form the IG's terminology-cleanup backlog for the
next ballot cycle.
