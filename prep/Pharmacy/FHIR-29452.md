## Summary

The reporter notes that a hyperlink in the US Medication FHIR Implementation
Guide (formerly the US Prescription Drug Monitoring Program / PDMP on FHIR
IG) is broken: specifically, the link associated with the phrase
"Meaningful Use 2015 §170.302(d)" no longer resolves. The request is a
straightforward editorial fix — repair the hyperlink so the cited ONC EHR
certification requirement is reachable from the IG narrative. The ticket
was originally resolved "Considered for Future Use" in January 2023 because
the IG had been withdrawn, then reopened in November 2023 in conjunction
with PSS-2225 (PDMP on FHIR), and was subsequently resolved "Not
Persuasive" with the rationale that the new IG does not reference Meaningful
Use requirements at all. No cross-referenced Jira tickets, Zulip
conversations, or GitHub items were surfaced for this issue.

## Proposed Dispositions

### Disposition A: Accept As Requested

#### Proposal

Restore a working citation for the ONC EHR certification criterion
historically referred to in the IG. Concretely: in the relevant narrative
page of the US Medication FHIR Implementation Guide (successor to the
withdrawn US PDMP on FHIR IG), replace the broken
"Meaningful Use 2015 §170.302(d)" hyperlink with the current canonical
URL for that criterion — the ONC Health IT Certification Program rule
text on healthit.gov / the Federal Register entry for 45 CFR §170.302(d)
(electronic prescribing) — and verify the link via the IG build's link
checker. If the criterion text is no longer available at a stable ONC URL,
substitute a citation to the equivalent §170.315(b)(3) e-Prescribing
criterion (the 2015 Edition successor) and update the surrounding prose
accordingly.

#### Justification

The change is purely editorial, has no impact on conformance, and
preserves the regulatory provenance the original authors intended to
convey. Editorial repair of dead external links is consistent with normal
IG maintenance and avoids leaving readers with a citation they cannot
follow.

---

### Disposition B: Alternative Approach

#### Proposal

Rather than chasing a stable URL for an ONC criterion that has been
superseded, remove the hyperlink (and, if appropriate, the surrounding
"Meaningful Use 2015 §170.302(d)" reference) from the US Medication FHIR
IG entirely. Replace it with a brief plain-text note such as
"historically aligned with ONC e-Prescribing certification requirements"
and, if the project sponsor wants a citation at all, point at the current
ONC Certification Program landing page rather than a deep link to a
specific criterion.

#### Justification

The current resolution comment ("new IG does not reference Meaningful
Use requirements") indicates the workgroup has already decided the
Meaningful Use framing is not part of the rebooted IG's scope. Removing
the dangling reference is therefore consistent with the new IG's
editorial direction, eliminates the link-rot problem at its source, and
avoids re-introducing a citation the workgroup has chosen to drop.

---

### Disposition C: Decline

#### Proposal

Confirm the existing resolution of *Not Persuasive* and close the ticket
without further action. Rationale: the US Medication FHIR Implementation
Guide (the successor to the withdrawn US PDMP on FHIR IG) does not
reference Meaningful Use requirements, so there is no "Meaningful Use
2015 §170.302(d)" hyperlink in the current spec to repair.

#### Justification

This is precisely the disposition already recorded on the ticket
(resolved 2023-11-28, *Not Persuasive*, "new IG does not reference
Meaningful Use requirements"). The defect described — a broken hyperlink
in narrative that no longer exists — is moot in the current IG. Spending
further editorial cycles on a citation the workgroup has consciously
removed offers no implementer benefit, and the original IG that contained
the broken link has been withdrawn.

---

### Recommendation

**Recommended disposition:** C

The ticket has already been resolved *Not Persuasive* on the basis that
the rebooted US Medication FHIR IG no longer references Meaningful Use
certification criteria, which makes the original defect (a broken link
to §170.302(d)) moot in the current spec. Reopening the ticket to chase
a working URL (Disposition A) would re-introduce a regulatory citation
the workgroup has chosen to drop, and Disposition B largely codifies a
state the new IG is already in. The cleanest outcome for the workgroup
is to confirm the existing resolution and close the ticket; if a reviewer
spots a residual mention of "Meaningful Use" anywhere in the current IG
during ballot review, that should be filed as a fresh, narrowly-scoped
editorial ticket against the specific page rather than reanimating this
one.
