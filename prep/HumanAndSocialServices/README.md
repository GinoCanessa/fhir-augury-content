#### Topic: HSDS data-model alignment

These tickets both target alignment between the Human Services Directory
(FHIR) IG and the upstream Open Referral HSDS source specification. One
flags a structural gap in how service-level contact information is carried
relative to HSDS, while the other tracks the broader set of HSDS 3.0
breaking changes (address merging, attribute renames, the new
`organization_identifier` table) that the IG's mappings and
`Organization.identifier` slicing must catch up to.
