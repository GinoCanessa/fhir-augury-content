#### Topic: OID and URI registry governance

These tickets all concern how HL7 OIDs (and, for FHIR-31621, the
parallel URI namespace) are registered, kept in sync between the FHIR
specification pages and the HL7 OID Registry / THO, and governed
across V2/V3/CDA/FHIR. They range from a single missing OID on a
specification page (FHIR-26384), to an unregistered value-set OID
(FHIR-26396), up to a broad governance ask for unified, version-aware
OID↔URI administration under TSMG/THO/UTG (FHIR-31621).
